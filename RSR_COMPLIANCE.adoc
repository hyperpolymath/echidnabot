== Language Tier Classification

=== Tier 1 (preferred)
* Rust ✓ (primary application code)
* Idris2 ✓ (ABI definitions — `src/abi/`)
* Zig ✓ (FFI scaffold — `ffi/zig/`, see link:ABI-FFI-README.md[ABI-FFI-README.md])
* AffineScript (no use in this repo yet; planned for future config DSLs)
* Ada, Haskell — N/A

=== Tier 2 (acceptable)
* Nickel — config snapshots in `config/`
* Guile Scheme ✓ — `guix.scm` packaging

Nix is **deprecated estate-wide** as of 2026-06-01 — `flake.nix` /
`flake.lock` removed from this repo on the same date; Guix is the sole
packaging primary going forward.

=== Banned (per estate policy 2026-05-25)
* TypeScript / JavaScript — use AffineScript
* Node.js / npm / Bun / pnpm / yarn — use Deno
* Go — use Rust
* Python — outlawed estate-wide (SaltStack-only carve-out, none here)
* Java / Kotlin / Swift — use Tauri / Dioxus
* Jekyll — use casket-ssg
* Dockerfile — use Containerfile

== Exemptions

[cols="2,3"]
|===
|Exemption |Justification

|`echidnabot-mcp` outside BoJ
|Documented carve-out in `.machine_readable/6a2/AGENTIC.a2ml`
 `[exceptions.boj-only-mcp]`. Sunset: when BoJ cartridge supports the
 double-loop feedback protocol.

|Dogfood proofs include intentional failures
|`proofs/coq/admitted_stub.v` and `proofs/lean/sorry_stub.lean` are
 fixtures used to exercise the failure path of CI. Bot self-mode is
 `advisor` (not `regulator`) to avoid blocking own merges.

|Standalone repo + monorepo dual-truth
|echidnabot is registered as a submodule in `hyperpolymath/echidna` but
 checked in as regular files. Acknowledged drift; fix path tracked in
 `STATE.a2ml` `[known-drift]`.

|`Cargo.toml` path dep on `../../shared-context`
|Standalone-clone builds fail at dep resolution. Acknowledged; tracked
 in issue #18. Working build requires the gitbot-fleet layout.
|===
